# Quadcopter

Repository for my quadcopter personal project. More details to come...

## Pinout for Arduino quadcopter

### Gyroscope
VCC -> 5V
GND -> GND
SCL -> A5 (on Arduino UNO)
SDA -> A4 (on Arduino UNO)

### Receiver
CH1[0] -> D8
CH2[0] -> D9
CH3[0] -> D10
CH4[0] -> D11
BAT[1] -> 5V
BAT[2] -> GND
